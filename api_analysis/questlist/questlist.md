可以合理推斷，
api_state 為 1 的時候，代表任務有顯示出來，但沒有被選取
為 2 的時候，代表有被選取中
為 3 的時候，代表可以領獎？



- **api_page_count** 總共有幾頁任務
- **api_disp_page** 目前顯示的是第幾個頁面
- **api_list**
  - **api_no** 任務的 server 端 id
  - **api_category** 任務類型，待研究，我猜是以「出擊、工廠、遠征、演習」等區分
  - **api_type** 任務種類，待研究，我猜是以「一次性任務、每日任務、每周、每月」等區分
  - **api_state** 任務狀況，
     - 1 代表任務有顯示出來，但沒有選取
     - 2 代表目前有被選取，會顯示”遂行中”
     - 3 代表任務已完成，可以領獎
  - **api_title** 任務名稱
  - **api_detail** 任務敘述
  - **api_get_material** 任務報酬，依序為燃料、彈藥、鋼材、機材
  - **api_bonus_flag** 待研究，我猜是除了資源外，後續會送家具箱那些?
  - **api_progress_flag** 目前進度，1 代表 50%，2 代表 80%
