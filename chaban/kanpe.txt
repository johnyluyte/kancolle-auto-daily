# VPN

這個東西在我當計網概助教的時候，有很多同學來問過我

老師在講你沒有在聽，助教講完你考試完就忘記了

不過沒關係，我們只需要知道 VPN 怎麼用就好了

我的電腦是 MAC，使用的 VPN client 是 tunnelblick

用 vimium 開啟 gatevpn
尋找日本伺服器
下載 configure file

我們使用的是免費的 VPN，免費的壞處就是 VPN server 不穩定，有可能今天連上，明天就連不上了

因此切換 VPN server 是很常見的情形

常常要設定新的 configure file 其實也蠻麻煩的

所以我就寫了一個簡單的小程式，

他會去 parse 這些 VPN server 的列表，過濾出我們想要的，然後自行安裝設定檔

示範 fetch.py (alias gatevpn?)
先全部 N
再一次，選 Y
小心密碼別打錯，下載並安裝 configure file
